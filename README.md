# TrafficSimulation

This is a traffic simulation application. 

The basis of the application belongs to HKUST-OCTAD-LAB at https://github.com/HKUST-OCTAD-LAB/AirTrafficSim, which can be used and modified under the GNU GENERAL PUBLIC LICENSE Version 3.

The basic AirTrafficSim application by HKUST-OCTAD-LAB has been heavily modified to support real-time traffic simulation, visualization via an X-Plane Connector bridge, ground network movement, and a ground model (etc.).

The X-Plane Plugin and connector have been developed by me at a full stage.

The repo contains the conda environment config file, the AirTrafficSim project without large data files (navigation, .dat, .csv etc.), and the X-Plane Plugin (ATS_XPlane_Bridge)
