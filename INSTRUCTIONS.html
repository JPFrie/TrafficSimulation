Traffic Simulation v0.1 Instructions


How to run the application

The TrafficSimulation Environment uses two main architectures - one python-based webserver and one X-Plane plugin. To start both of them, do the following:

TrafficSimulation:

  1. Start DA 1 PC and open a terminal
  2. Type in terminal "conda activate airtrafficsim" to activate and open the corresponding anaconda environment
  3. Check that the conda environment is active - in front of sim@devbox should be a (airtrafficsim) tag
  4. Start the webserver environment with the command "airtrafficsim". You'll be asked for the IP of the X-Plane PC. Enter the IP address of DA4 SIMZERO (normally 10.18.20.22, in case check with ifconfig on the SIMZERO)
  5. Open a webbrowser and enter the link "localhost:6111". You should now see a world map.
  6. Make sure that the X-Plane plugin is already running at this step.
  7. To open a scenario, click the "Simulation" button in the lower menu bar. You can select the specific scenario in the popup window and by clicking on it, The scenario takes some time to load since it pre-loads the X-Plane navigation database and caches it internally. Currently the development scenario is MUCTEST
  8. (WIP): To modify aircraft parameters, click the green button "A/C Controls". It is currently still work in progress and will be fully functional with the next development version.
  9. If you want to stop the application, go to the terminal and press CTR+C two times to get back access to the normal terminal. If you want to deactivate the anaconda environment, type in the command "conda deactivate" and you are back in the base linux environment (since python packages for airtrafficsim environment are stored locally in the conda environment and may not available in the basic linux system).

X-Plane 11/12 Plugin:

  1. Start DA 4 SIMZERO PC and start X-Plane 12 with all the simulation framework.
  2. In X-Plane, hover over the top control bar and head to Plugins->AI_XPlane_Bridge->Start/Stop and click it. You can confirm that it runs when this menu option has a dot before the "Start/Stop".
  3. Make sure that you start the X-Plane Plugin BEFORE you load a scenario in the Traffic Simulation.

How to create a new scenario:

  1. Open a folder or terminal and navigate to ?/home/sim/anaconda3/envs/airtrafficsim/python3.10/site-packages/airtrafficsim/data/environment
  2. Copy the "OpenApDemo.py file and open it.
  3. Confirm in the init method that "performance_mode" is set to "OpenAP". BADA performance model will be accessible in the future.
  4. Open MUCTEST.py scenario and get familiar with the structure of how to add and modify aircraft (parameters). A detailed description will be added in the future.
